Javascript Fundamentals

Variables

In Javascript there are three keywords used for variables. These are var, let and const. One great way to think of a variable is to think of it as a binding. They grasp and hold onto the value. They are like a container.
The keyword var was used in pre-2015 JavaScript and is short for variable. Const is short for constant which means the value will never change as long as it lives.
The keyword let is used if it is a variable(binding) that you need to mutate or reassign later on(not within the same block).

var x = 5;
var y = 6;
var z = x + y;

{
let x = "John Doe";
let x = 0;
}

const PI = 3.141592653589793;

PI = PI + 10; // This will also give an error

Operators

The assignment operator (=) assigns a value to a variable.
The addition (+), multiplication (*), subtraction (-), and division (/) symbols are called arithmetic operators.
When the operators appear together without parentheses, the order is determined by precedence. Multiplication and division come before addition and subtraction but when in doubt just add parentheses(which have the highest precedence).
The remainder (%) operator has the same precedence as multiplication and division. X % Y is the remainder of dividing X by Y.
The increment operator (++) adds by 1, and the decrement operator (--) subtracts by 1.
Unary operators are words that produce a string value naming the type of value you give it.

let x = 5; // assign the value 5 to x
let y = 2; // assign the value 2 to y
let z = x + y; // assign the value 7 to z (5 + 2)

Assignment Operators
= operator is the same as x = y
+= operator is the same as x = x + y
-= operator is the same as x = x - y
*= operator is the same as x = x * y
/= operator is the same as x = x / y
%= operator is the same as x = x % y
**= operator is the same as x = x ** y

The + operator can also be used to add(concatenate) strings.
let text1 = "John";
let text2 = "Doe";
let text3 = text1 + " " + text2;
// result will be "John Doe"

The += assignment operator can also be used to add (concatenate) strings:
let text1 = "What a very ";
text1 += "nice day";
// result will be "What a very nice day"

Adding Strings and numbers:
Adding two numbers, will return the sum, but adding a number and a string will return a string.
let x = 5 + 5; // result is 10
let y = "5" + 5; // result is 55
let z = "Hello" + 5; // result is Hello5

See a few operators examples here

JavaSvript has 3 logical operators: And (&&), Or (||), and Not (!)
These logical operators can be used to "reason" about Booleans. The and operator (&&) is a binary operator which means it needs two values. It's result is true only if both values given are true.
The or operator (||) is also a binary operator and produces true if either of the values given to it are true.
The not (!) operator is a unary operator that flips the value given. So !true produces false.
With these, || has the lowest precedence, then &&, then the comparison operators, then the rest.

let x = 6;
let y =3;
(x < 10 && y> 1) is true
(x == 5 || y == 5) is false
!(x == y) is true

Ternary operators require three values. It is written with a question mark and a colon. The value on the left of the question mark picks which of the other two values will come out. When it's true it chooses the middle value, and when it is false it chooses the value on the right.

let voteable = (age < 18) ? "Too young" :"Old enough";

In order to make a decision, your code checks the currect status of the script. This is commonly done by comparing two values using a comparison operator which returns a value of true or false.

Conditional Statements

The if statement evaluates (or checks) a condition. If the condition evaluates to true, any statements in the subsequent code blocks are executed.
The if...else statement checks a condition. If it resolves to true the first code block is executed. If the condition resolves to false the second code block is run instead.
A switch statement starts with a variable called the switch value. Each case indicates a possible value for this variable and the code that should run if the variable matches that value.
At the end of each case is the break keyword. It tells Javascript that is has finished with the switch statement and to proceed to run any subsequent code that appears after it.
The default keyword specifies the code to run if there is no case match.

if (hour < 18) {
greeting = "good day";
}

if (hour < 18) {
greeting = "Good day";
} else {
greeting = "Good evening";
}

switch(expression) {
case x:
// code block
break;
case y:
// code block;
break;
default:
// code block
}

Loops

Loops are handy, if you want to run the same code over and over again.
The for loop - loops through a block of code a number of times. The parentheses after a for keyword must contain two semicolons. The part before the first semicolon initializes the loop, the second part checks whether the loop must continue, the final part updates the state of the loop after every iteration.

for (statement 1; statement 2; statement 3) {
// code block to be executed
}

The do while loop - is a little different. The statements in the code block come before the condition. This means that those statements are run once whether or not the condition is met.

do {
//code block to be executed
}
while (condition);

Functions

Functions let you group a series of statements together to perform a specific task. You can reuse a function if parts of a script repeat the same task.
A function can have parameters listed inside the parentheses after the function name. Inside the function the arguments (parameters), behave as local variables.
You always have to call a function to run it.

function name(parameter1, parameter2, parameter3) {
// code to be executed
}

JS Objects General Overview

Objects are like variables but they can contain many values.
The values are written as name:value pairs, separated by a colon. These are called properties.
Property bindings are similar to variable bindings. They grasp values.
With objects you can change their properties, causing a single object value to have a different content at different times.
Objects can also have methods. Methods are actions that can be performed on objects.

Example: const phone = {type: "iPhone", model: "XR", color: "pink"}; //This example shows an object with properties
Example of accessing an objects properties: objectName.propertyName or objectName["propertyName"]

Method syntax: objectName.methodName();
Example of method: phone.dialNumber();

In a function definition, this refers to the "owner" of the function. Using the above phone object example, this.model, means the model property of the phone object.

Array Object

Arrays let you store multiple values in a single variable. This is especially useful if you're working with a list or a set of values that are related to each other. Array's are objects but in JavaScript it is best described as an array.
They are also helpful when you don't know how many items a list will contain because when you create an array, you don't have to specify how many values it will hold.
When you create an array you give it a name(like any other variable) and the values are assigned inside a pair of square brackets and each value is separated by a comma. The values don't have to be the same data type.
Array elements always start with 0. The first element is always at 0.
Syntax for creating an Array
const array_name = [item1, item2, ....];

const cars = ["Saab", "Volvo", "BMW"];
let x = cars[0]; // x = "Saab"

You can also create an array, and then provide the elements:
const cars = [];
cars[0] = "Saab";
cars[1] = "Volvo";
cars[2] = "BMW";

JavaScript variables can be objects. Arrays are special kinds of objects.
Because of this, you can have variables of different types in the same Array.
You can have objects in an Array. You can have functions in an Array. You can have arrays in an Array:
myArray[0] = Date.now;
myArray[1] = myFunction;
myArray[2] = myCars;

Array's have built-in properties and methods:
cars.length // Returns the number of elements
cars.sort() // Sorts the array

Accessing the first element in an array:
const fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits[0]; // Returns "Banana"

Accessing the last element in an array:
const fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits[fruits.length - 1]; // Returns "Mango"

Arrays have methods. Their property names are index numbers. You will often see arrays used to store complex data(including other objects).

push() - Adds one more item to the end of an array and returns number of items in it.
unshift()- Adds one more item to the start of an array and returns new length of it.
pop() - Removes last element from array and returns the element.
shift() - Removes first element from array and returns the element.
forEach() - Executes a function once for each element in an array.
some() - Checks if some elements in an array pass a test specified by a function.
every() - Checks if all elements in array pass a test specified by a function.
concat() - Creates new array containing this array and other arrays/values.
filter() - Creates new array with elements that pass a test specified by a function.
sort() - Reorders items in array using a function(called a compare function).
reverse() - Reverses order of items in array.
map() - Calls a function on each element in array and creates new array with results.

String Object

JavaScript strings are used for storing and manipulating text. A string is zero or more characters written inside quotes.
Example:
let text = "John Doe";
You can use single or double quotes. You can use quotes inside a string, as long as they don't match the quotes surrounding the string:
let answer2 = "He is called 'Johnny'";

Strings also have methods:
length - Returns number of characters in the string.
toUpperCase() - Changes string to uppercase characters.
toLowerCase() - Changes string to lowercase characters.
charAt() - Takes an index number as a parameter, and returns the character found at that position.
indexOf() - Returns index number of the first time a character or set of characters is found at that position.
lastIndexOf() - Returns index number of the last time a character or set of characters is found within the string.
substring() - Returns characters found between two index numbers where the character for the first index item number is included and the character for the last index number is not included.
split() - When a character is specified, it splits the string each time it is found, then stores each individual part in an array.
trim() - Removed whitespace from start and end of a string.
replace() - Like find and replace, it takes one value that should be found, and another to replace it (by default, it only replaces the first match it finds).
slice() - Extracts a part of a string and returns the extracted part in a new string. It takes 2 parameters - the start position, and the end position. if a parameter is negative, the position is counted from the end of the string. If you omit the second parameter, the method will slice out the rest of the string.
substr() - Similar to slice() but the second parameter specifics the length of the extracted part.
concat() - Joins two or more strings.

Strings also have escape characters:
\' Gives the result ' (single quote)
\" Gives the result " (double quote)
\\ Gives the result \ (backslash)
\b Gives a backspace
\f Gives a form feed
\n Gives a new line
\r Gives a carriage return
\t Gives a horizontal tabulator
\v Gives a vertical tabulator

Strings are normally primitive values in JavaScript, created from literals:
let firstName = "John";
But strings can also be defined as objects with the keyword new:
let firstName = new String("John");

Javascript - Web Browser - Web Document Interaction

Browser Objects

The Browser Object Model allows Javascript to "talk to" the browser. It contains objects that represent the currect browser window or tab. It contains objects that model things like browser history and the device's screen.
The topmost object is the window object, which represents the current browser window or tab. It's child objects represent other browser features:
window - current browser window or tab.
Example: window.innerHeight - the inner height of the browser window(in pixels).
document - current web page.
Example: document.getElementById("header");
history - contains the browsers history.
Example: history.back() - same as clicking back in the browser.
location - URL or current page.
Example: window.location.href - returns the href(URL) of the current page.
navigator - information about browser.
Example: navigator.cookieEnabled - returns true if cookies are enabled, otherwise false.
screen - device's display information.
Example: screen.width - returns the width of the visitor's screen in pixels.
popup alert - JavaScript has three kinds of popup boxes: Alert box, Confirm box, and Prompt box.
Example: window.alert("hello World!") - gives an alert on the screen. This method can also be written without the window prefix.
timing - allows execution of code at specified time intervals.These time intervals are called timing events.
Example: setTimeout(function, milliseconds) - executed a function, after waiting a specified number of milliseconds.
setInterval(function, milliseconds) - same as setTimeout(), but repeats the execution of the function continuously.
cookies - let you store user information in web pages.
Example: document.cookie = "username = John Doe";
